# push_swap

This repository contains a simplified push_swap project focusing only on argument parsing.

- Use your own `libft` placed at `./libft` (the `Makefile` expects `libft.a` at `libft/libft.a`).
- The project uses `include/` for headers and `src/` for sources.

## Setup

1. Clone or copy your `libft` to `./libft` and build it (`make` in `libft/`).
2. Run `make` in project root to build `push_swap`.

## Usage

- `./push_swap [numbers]`: Parses the arguments and prints them if valid.

## Project Structure

- `src/main.c`: Entry point that parses and prints numbers.
- `src/args.c`: Argument parsing and validation.
2. Run `make` in project root to build `push_swap`.
